=== TECHART 評量矩陣 — Netlify 部署包 ===

上傳前請先確認：

1. 將您的 LOGO.png 放入此資料夾（與 index.html 同層）
   - 檔案名稱必須是：LOGO.png（大寫）
   - 建議尺寸：高度 48px ~ 96px 以上的透明 PNG

2. 部署方式（選一）：
   A. 直接拖拉整個資料夾到 Netlify（推薦）
      → 前往 app.netlify.com → Sites → 拖曳整個資料夾
      
   B. 壓縮成 .zip 後上傳
      → 將資料夾內所有檔案選取壓縮（不要壓縮資料夾本身）
      → 上傳到 Netlify → Deploy manually

3. Firebase Console 確認：
   → Authentication → Sign-in method → 匿名登入 → 啟用
